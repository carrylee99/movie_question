# 影视剧知识问答 - 你真的看懂了吗？ V2.0

一个精美的影视剧知识问答网页，测试你对经典剧集的理解程度！每次挑战都有不同的题目组合。

## 🎭 功能特色

### 📺 三大经典选题
- **甄嬛传** 👑 - 宫斗经典，测试你的心机洞察力 (50道题库)
- **大明王朝1566** 🏛️ - 权谋巅峰，考验你的官场博弈理解 (50道题库)
- **知否知否** 🌸 - 古代生活，检验你对细节的观察 (50道题库)

### 🎯 核心功能
- **随机题目**：每个选题从50道题中随机选择10道，每次挑战都不同
- **搞怪选项**：每道题至少包含一个幽默搞怪的错误选项
- **实时反馈**：答题后立即显示正确答案和详细解释
- **等级评定**：根据得分给出不同的评级和评价
- **精美动效**：现代化暗色主题 + 丰富的动画效果

### 🏆 评级系统
- **🏆 剧迷大师** (90%+) - 你就是行走的剧集百科全书！
- **🎭 资深观众** (75%+) - 对剧情理解相当深刻
- **📺 合格观众** (60%+) - 基本看懂主线剧情
- **🤔 入门观众** (60%以下) - 建议重新专心观看

## 🚀 使用方法

1. 打开 `quiz.html` 文件
2. 选择你想挑战的剧集
3. 系统随机选择10道题目
4. 认真阅读题目，选择答案
5. 查看实时反馈和解释
6. 完成所有题目获得评级
7. 可重复挑战，每次题目都不同
