# 痛风智护网 (GoutCare Pro) - V1.0

专为痛风患者打造的全周期健康管理工具，纯前端实现，数据本地存储。

## 功能特性

### ✅ V1.0 已实现功能

- **🏠 首页**
  - 健康阶段选择（预防期/缓解期/初期/急性发作期）
  - 快速操作入口
  - 应急指导弹窗

- **🍎 食物嘌呤查询**
  - 60+ 种常见食物嘌呤数据
  - 模糊搜索功能
  - 按嘌呤等级过滤（高/中/低）
  - 今日嘌呤摄入量统计
  - 一键添加到饮食记录

- **📊 健康记录**
  - 尿酸值记录与历史查看
  - 痛风发作记录（时间/部位/疼痛等级/诱因）
  - 今日饮食记录管理

- **📋 阶段指导**
  - 根据用户选择的健康阶段提供个性化指导
  - 饮食建议、生活习惯、用药指导等

- **🔧 实用工具**
  - 尿酸单位换算（μmol/L ↔ mg/dl）
  - 疼痛等级评估（1-10级可视化）

## 设计亮点

- **🎨 阶段主题色彩**：界面背景根据健康阶段自动切换颜色
- **📱 响应式设计**：完美适配手机、平板、电脑
- **💾 本地存储**：所有数据存储在浏览器本地，保护隐私
- **🚀 即开即用**：无需注册登录，打开即可使用

## 技术实现

- **前端技术**：原生 HTML5 + CSS3 + JavaScript ES6+
- **数据存储**：localStorage 本地存储
- **响应式**：CSS Grid + Flexbox
- **兼容性**：支持现代浏览器

## 使用方法

1. 直接打开 `index.html` 文件
2. 选择您当前的健康阶段
3. 开始使用各项功能
